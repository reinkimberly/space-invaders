# space-invaders
This project is a Java recreation of Space Invaders built using the Model-View-Controller (MVC) design pattern, where the player controls a ship that moves horizontally, shoots aliens, and progresses through increasingly difficult levels while managing lives and score. The GameModel handles all game logic and state, including player movement, alien behavior, bullets, collisions, shields with health, score tracking, and level progression up to level 10 with increasing speed and difficulty, while also determining when the game is over or a level is complete. The GameView is responsible for rendering all visual elements, including the player, aliens, bullets, shields (colored based on health), score, lives, high score, and level, as well as displaying centered messages such as “Game Over,” “Next Level,” and the final “Congratulations You Beat the Aliens!” screen. The GameController connects everything by creating the window, handling keyboard input (movement, shooting, and continuing between levels), and running the game loop with a timer that updates the model and repaints the view, including pausing between levels until the player presses “C” to continue. Additional features include destructible shields(with color for heath), dynamic difficulty that increases as aliens are destroyed, a high score display, and a level transition system that pauses gameplay and resets the board before advancing, creating a complete and interactive arcade-style experience.
